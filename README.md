# modterm

This is a GPU accelerated terminal emulator for Windows. It features VT100 terminal emulation, full ANSI (24-bit/256) color support, and a configurable glass-like UI. It uses VtNetCore for terminal emulation, and ConPTY for terminal device emulation.
